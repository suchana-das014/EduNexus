# EduNexus

EduNexus is a modern, smart school management platform designed to simplify the administration of educational institutions. It provides an intuitive interface to manage teachers, students, courses, and enrollments in a seamless and interactive way.

Built with **Laravel**, **Inertia.js**, **React**, and **Tailwind CSS**, EduNexus combines the power of a backend framework with a modern frontend experience to deliver fast, responsive, and fully functional school management software.

---

## Key Features

EduNexus is built for simplicity and efficiency. The main features include:

### Dashboard Analytics
- Quick overview of total students, teachers, courses, subjects, and enrollments.
- **Desktop:** All cards fully visible without scrolling.
- **Mobile:** Scrollable cards for a responsive experience.

### Student Management
- Add, update, and delete student records.
- Track grades, personal details, and enrollment history.

### Teacher Management
- Manage teacher profiles, subjects, and assigned courses.
- Quickly update or remove teacher information.

### Course Management
- Create courses and assign teachers.
- View course details and manage enrollments.

### Enrollment Management
- Enroll students in courses with ease.
- Keep track of enrollment dates and course participation.

### Responsive Design
- Fully optimized for desktop, tablet, and mobile devices.
- Tables are adaptive: scrollable on small screens, fully visible on desktops.

### User-Friendly UI
- Clean, modern interface with interactive cards, modals, and tables.
- Quick actions for adding, editing, and deleting records.

---

## Tech Stack

- **Backend:** Laravel 10 (PHP 8.1+)
- **Frontend:** React.js + Inertia.js
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Database:** MySQL / MariaDB
- **Routing & State Management:** Laravel + Inertia
